## Linux Note


1. grep keyword
2. ls ./* | grep test (search file title, not content)
3. 常用
    - -i (不區分大小寫)
    - -n (列出第幾行)
    - -r (搜尋子目錄)
4. regular expression
    - "^a" a 開頭
    - "a$" a 結尾
    - "^[ab]" a or b 開頭
    - "^ab*" a 開頭, b 出現任意次
    - "^ab?" a 開頭, b 出現 0 or 1
    - "^ab+" a 開頭, b 出現一次以上
    - "ab|cd" 出現 ab or cd
    - "<abc>" 準確搜尋
